# 📦 Sistema de Inventario

Aplicación desarrollada en PHP y MySQL para la gestión de productos, marcas y clientes.

## Funcionalidades

* Registro de clientes
* Registro de productos
* Registro de marcas
* Relación entre productos y marcas

## Tecnologías

* PHP
* MySQL
* HTML
* CSS

## Estado del proyecto

En desarrollo (se agregarán módulos de ventas y stock)
